# My-Portfolio
A Website designed fully by me to display my portfolio, resume, and general information about me.

## Technologies
### Frontend
- React.js
- material-ui for any icons used
- Sass for styling and responsive design
### Backend
- Python
- Flask
- Flask-Mail
